◆ banded_invmul()

subroutine numa__linalg::banded_invmul ( double precision, dimension(:,:), intent(inout) a,
double precision, dimension(:), intent(inout) bx,
integer, intent(in) nl,
integer, intent(in) nu )

Solves a linear system a*x == b for x with a banded matrix a.

Banded matrix a is packed into a mostly sparse rectangular array. For example, this data structure: 

nl = 2  ! number of lower bands
nu = 1  ! number of upper bands
n  = 5  ! size of x
allocate(a(nl+nu+1, n))
a(:, 1) = [0, 0, 5, 2]
a(:, 2) = [0, 3, 6, 2]
a(:, 3) = [2, 3, 7, 2]
a(:, 4) = [2, 3, 8, 2]
a(:, 5) = [2, 3, 3, 0]

represents this dense matrix in MATLAB/Scilab layout: 

a = [
    5 2 0 0 0
    3 6 2 0 0
    2 3 7 2 0
    0 2 3 8 2
    0 0 2 3 3
]
Parameters
[in,out]aBanded matrix on input, factored on output
[in,out]bxRHS b on input, x on output
[in]nlNumber of lower non-zero bands below diagonal
[in]nuNumber of upper non-zero bands above diagonal
